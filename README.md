## 🧠 Challenge Description
From Data to Insight: การใช้ประโยชน์จากข้อมูล Open Data

การแข่งขันนี้มีเป้าหมายเพื่อส่งเสริมการเรียนรู้การใช้ประโยชน์จาก Open Data ผ่านกระบวนการวิเคราะห์ข้อมูลอย่างเป็นระบบ ตั้งแต่การเตรียมข้อมูล การสำรวจข้อมูล ไปจนถึงการสกัด Insight ที่มีความหมายต่อการตัดสินใจ

ผู้เข้าร่วมจะได้ฝึกใช้ทักษะด้าน Data Preparation, Exploratory Data Analysis (EDA), และ Data Visualization เพื่อค้นหารูปแบบ แนวโน้ม และประเด็นสำคัญที่ซ่อนอยู่ในข้อมูล พร้อมนำเสนอผลลัพธ์ผ่าน Dashboard หรือ Visualization ที่สื่อสารข้อมูลได้อย่างชัดเจน

### 🎯 Objective
วิเคราะห์ข้อมูล Open Data ที่กำหนด เพื่อค้นหา Insight ที่มีความหมาย และนำเสนอผลการวิเคราะห์ผ่าน Visualization หรือ Dashboard ที่ช่วยให้เข้าใจข้อมูลได้ง่ายและสามารถนำไปใช้ประกอบการตัดสินใจได้

### 📌 Task
ผู้เข้าร่วมต้องเลือกชุดข้อมูล จากชุดข้อมูลที่กำหนดให้ และดำเนินการวิเคราะห์ข้อมูลตามขั้นตอนดังต่อไปนี้

#### 1️⃣ Data Preparation
ทำความเข้าใจโครงสร้างข้อมูล
ทำความสะอาดข้อมูล (Data Cleaning)
จัดรูปแบบข้อมูลให้พร้อมสำหรับการวิเคราะห์

#### 2️⃣ Exploratory Data Analysis (EDA)
สำรวจลักษณะของข้อมูล
วิเคราะห์แนวโน้ม รูปแบบ หรือความสัมพันธ์ของข้อมูล
สร้างกราฟหรือ Visualization เพื่อช่วยทำความเข้าใจข้อมูล
#### 3️⃣ Insight Discovery
ระบุ ประเด็นสำคัญหรือ Insight ที่น่าสนใจ
อธิบายความหมายของข้อมูลที่ค้นพบ
ตอบคำถามจากโจทย์ของชุดข้อมูลที่เลือก


### 📊 Expected Output
ผู้เข้าร่วมต้องส่งผลงานประกอบด้วย
1) Data Storytelling Notebook (ภายในวันที่ 20 มีนาคม 2569)
แสดงขั้นตอนการวิเคราะห์ข้อมูล ตั้งแต่การเตรียมข้อมูล การทำความสะอาดข้อมูล การสำรวจข้อมูล การสร้างกราฟเพื่อหา Insight และตอบคำถามจากโจทย์
มีโค้ดและคำอธิบายการวิเคราะห์อย่างชัดเจน
2) คลิป VDO นำเสนอผลการวิเคราะห์ข้อมูล Data Storytelling (ภายในวันที่ 31 มีนาคม 2569)
• ความยาวไม่เกิน 1 นาที ส่งเป็น Link เช่น Tiktok, Reel, Youtube, file
• มีการนำเสนอข้อมูลประเด็นที่ได้จากการวิเคราะห์ข้อมูล ครบ 3 ส่วน เนื้อหา (context) → insight → so what (สรุปผลและสิ่งที่ต้องดำเนินการต่อ)
